# 🌦️ Weather App

A simple and responsive Weather App built using **HTML**, **CSS**, and **JavaScript** that fetches live weather data using the **OpenWeatherMap API**.

## 🚀 Features

*  Search weather by city name
*  Display temperature in Celsius
*  Show humidity percentage
*  Display wind speed
*  Dynamic weather icons
* Responsive design for different screen sizes
*  Error message for invalid city names

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder:

```bash
cd weather-app
```

3. Open `index.html` in your browser.

## 🔑 API Setup

This project uses the OpenWeatherMap API.

1. Create an account on OpenWeatherMap.
2. Generate your API key.
3. Replace the API key in `Weather.js`:

```javascript
const apikey = "e68b6e3ed2fdd8a21479e6a6d19a982d";
```

##  How It Works

* User enters a city name.
* JavaScript sends a request to the OpenWeatherMap API.
* Weather data is fetched and displayed dynamically on the page.
* Weather icons change according to the current weather condition.
